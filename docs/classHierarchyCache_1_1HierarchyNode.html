◆ getReverseReachableInheritanceSubgraph()

std::map< Id, std::vector< std::pair< Id, Id > > > HierarchyCache::HierarchyNode::getReverseReachableInheritanceSubgraph ( ) const

Determine the reversed subgraph of all nodes and edges that are reachable from this node.

The subgraph is represented by a map that maps a node ID t to a set of pairs where each pair consists of a node ID s and and edge ID e such that e refers to an edge from s to t. Note that the mapping is reversed compared to the edges.
