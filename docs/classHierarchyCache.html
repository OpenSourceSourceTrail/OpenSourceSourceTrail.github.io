Determine nodes and edges from which a specific node can be reached in a reversed graph.

A reversed graph can be produced by HierarchyNode::getReverseReachableInheritanceSubgraph().

Parameters
[in]nodeIdID of the target node.
[in]reverseGraphThe reversed graph.
[out]nodesThe nodes from which the node nodeId can be reached.
[out]edgesThe edges from which the node nodeId can be reached.
Precondition
The arguments for nodes and edges must be provided empty.
